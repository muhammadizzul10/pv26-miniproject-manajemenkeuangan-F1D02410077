## Smart Expense Trancker
Smart Expense Tracker adalah aplikasi desktop manajemen keuangan pribadi yang dibangun menggunakan Python dengan framework PySide6 dan database SQLite. Aplikasi ini dirancang untuk membantu pengguna mencatat, mengelola, dan memantau transaksi keuangan seperti pemasukan dan pengeluaran secara lebih terstruktur dan mudah dipahami.

Aplikasi berjalan sepenuhnya secara offline dan menyimpan seluruh data di lokal menggunakan file database SQLite, sehingga data tetap aman dan tersimpan meskipun aplikasi ditutup. Smart Expense Tracker juga menyediakan ringkasan keuangan secara real-time berupa total pemasukan, pengeluaran, dan saldo.

Selain itu, aplikasi ini dilengkapi dengan fitur CRUD (Create, Read, Update, Delete) untuk mengelola data transaksi, serta tampilan tabel interaktif untuk memudahkan pengguna dalam melihat dan memilih data. Antarmuka aplikasi juga didukung dengan styling menggunakan QSS agar tampilan lebih modern dan nyaman digunakan.

Teknologi yang digunakan:

• Python 3: bahasa pemrograman utama
• PySide6: framework GUI untuk membangun antarmuka desktop
• SQLite: database ringan berbasis file untuk penyimpanan data lokal
• QSS: untuk styling antarmuka dari file eksternal
• Qt Widgets: komponen UI seperti tabel, button, dan dialog
